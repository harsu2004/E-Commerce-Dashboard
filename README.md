1. Project Title:- 
 E-Commerce Sales Dashboard

2. What the Dashboard Does & Why It Exists:- 
This project showcases an interactive E-Commerce Sales Dashboard that provides insights into overall sales performance, product demand, and customer purchasing behavior.

The dashboard is designed to:-
  - Monitor total sales, profit, and products sold
  - Analyze sales distribution across regions and categories
  - Identify popular payment methods
  - Track monthly sales trends

3. Key Technologies Used:- 
  - Power BI – for dashboard creation
  - Microsoft Excel– for dataset handling
  - Data Visualization Techniques – bar charts, line charts, KPI cards
  - Basic Data Analysis – aggregation, filtering, categorization

4. Key Highlights of the Dashboard:-

- KPI cards showing Total Sales, Total Profit, and Total Products Sold
- Sales by Region to compare geographical performance
- Sales by Category to identify top-selling product types
- Most Used Payment Methods analysis
- Monthly Sales Trend for time-based insights
- Clean and structured layout for easy understanding
